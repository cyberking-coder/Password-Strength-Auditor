# Password Strength Auditor

A Python-based tool that validates password strength using Regex and checks for compromised passwords using the HaveIBeenPwned API.

## Features
- Password policy validation
- Breach detection
- Strength scoring
- Secure k-Anonymity API usage

## Technologies
- Python
- Regex
- HaveIBeenPwned API

## Usage
```bash
python Task01_Password_Strength_Auditor.py
